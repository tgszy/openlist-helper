## 版本格式说明

本更新日志遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范，项目版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

### 变更类型

- **新增功能**：添加了新功能
- **修复的问题**：修复了bug
- **改进的功能**：对现有功能进行了优化
- **破坏性变更**：不兼容的API更改
- **废弃功能**：即将移除的功能
- **安全性**：修复了安全漏洞

### 如何更新

1. 从 GitHub Release 页面下载最新版本
2. 或使用 Docker 拉取最新镜像：`docker pull tgszy/openlist-helper:latest`
3. 或使用 Docker 拉取指定版本：`docker pull tgszy/openlist-helper:v1.0.0`
